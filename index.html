Largura do Passeio SP

Este mapa pretende ilustrar como a largura dos passeios comprometem a capacidade dos pedestres em praticar o distanciamento social. As larguras são determinadas a partir da base de dados georeferenciados de São Paulo GeoSampa, cuja precisão e completude não foram verfificadas.

O mapa foi adaptado do projeto Sidewalk Widths NYC de Meli Harvey, para saber mais como o projeto foi desenvolvido visite a página do GitHub.

ATENÇÃO
Não use este mapa para tomar decisões que impactem sua saúde ou segurança. Outros importantes fatores como obstruções no passeio e intensidade do fluxo de pedestres não são considerados nos dados.
